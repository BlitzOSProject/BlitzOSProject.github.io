# Diamond

<div dir="rtl">

برنامه‌ای در KPL بنویسید که یک عدد فرد n به عنوان ورودی بگیرد و یک لوزی به قطر n چاپ کند.

برای گرفتن یک عدد به عنوان ورودی از InputInt استفاده کنید. مثلا:
<div dir="ltr">
```
n = InputInt()
```
</div>
</div>

### Example 1:

**Input:**

```
3
```

**Output:**

```
 *
***
 *
```

### Example 2:

**Input:**

```
7
```

**Output:**

```
   *
  ***
 *****
*******
 *****
  ***
   *
```
