# Triangle

<div dir="rtl">

برنامه‌ای با KPL بنویسید که 3 عدد ورودی بگیرد و چک کند که آیا مثلثی با این اضلاع می‌تواند وجود داشته باشد یا نه. در صورتی که وجود داشت ```Yes``` و در غیر این صورت ```No``` چاپ کنید.

</div>

### Example 1:

**Input:**

```
5
3
4
```

**Output:**

`Yes`

### Example 2:

**Input:**

```
1
2
3
```
**Output:**

`No`
